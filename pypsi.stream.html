class pypsi.stream.AnsiStream(stream, ansi_mode=0, width=80)[source]¶

Bases: object

Enhanced file stream wrapper that includes the ability to:

  • redirect output to a new stream, while keeping a stack of redirections
  • reset to the initial stream (cancel all redirects)
  • format an ansi string

The pypsi.shell.Shell.bootstrap() wraps both sys.stdout and sys.stderr in an AnsiStream instance.

Parameters:
  • stream (file) – the initial stream to wrap (should be either sys.stdout or sys.stderr)
  • ansi_mode (int) – the mode to use when checking if the active stream supports ansi escape codes
  • width (int) – the streams width, which is used by pypsi_print() and pypsi.format.wrap_line()
ansi_format(tmpl, **kwargs)[source]¶

Format a string that contains ansi code terms. This function allows the following string to be the color red:

sys.stdout.ansi_format("{red}Hello, {name}{reset}", name="Adam")

The pypsi.format.AnsiCodesSingleton.codes dict contains all valid ansi escape code terms. If the current stream does not support ansi escape codes, they are dropped from the template prior to printing.

Parameters:tmpl (str) – the string template
ansi_format_prompt(tmpl, **kwargs)[source]¶

Format a string that contains ansi code terms. This function allows performs the same formatting as ansi_format(), except this is intended for formatting strings in prompt by calling AnsiCode.prompt() for each code.

pypsi.stream.pypsi_print(*args, sep=' ', end='\n', file=None, flush=True, width=None, wrap=True)[source]¶

Wraps the functionality of the Python builtin print function. The pypsi.shell.Shell.bootstrap() overrides the Python print() function with pypsi_print().

Parameters:
  • sep (str) – string to print between arguments
  • end (str) – string to print at the end of the output
  • file (file) – output stream, if this is None, the default is sys.stdout
  • flush (bool) – whether to flush the output stream
  • width (int) – override the stream’s width
  • wrap (bool) – whether to word wrap the output
